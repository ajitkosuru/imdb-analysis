# 🎬 IMDb Movie Rating & Genre Trend Analysis

---

## 📌 Description

This project analyzes a dataset of **7,000+ movies** to uncover rating trends, genre distribution patterns, and evolving audience preferences.

The objective was to simulate a real-world business intelligence case where data-driven insights guide content production and investment strategies.

---

## 🛠 Technologies Used

- **SQL** – Data Cleaning & Aggregation  
- **Python** – Exploratory Data Analysis  
- **Pandas** – Data Manipulation  
- **Matplotlib / Seaborn** – Data Visualization  
- **Jupyter Notebook** – Analysis Environment  

---

## 🚀 Features

- Cleaned and standardized raw movie dataset  
- Genre-wise rating comparison  
- Pre-2010 vs Post-2010 trend analysis  
- Correlation analysis between variables  
- Visual dashboards for decision-making insights

## 🔎 The Process

### 1️⃣ Data Cleaning
- Handled null values  
- Fixed formatting inconsistencies  
- Standardized dataset structure  

### 2️⃣ Exploratory Data Analysis
- Analyzed genre distribution  
- Studied rating trends over time  
- Built correlation matrix  

### 3️⃣ Trend & Insight Analysis
- Compared Pre-2010 vs Post-2010 performance  
- Identified high-performing genres  
- Extracted business-driven insights  

---

## 📊 Key Insights

- **Drama and Action genres showed ~15% higher average ratings post-2010**
- Audience preference shifted toward story-driven and high-production films  
- Genre popularity patterns influence long-term content strategy
